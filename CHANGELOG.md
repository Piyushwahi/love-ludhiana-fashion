## [Unreleased]

### Added

- Project foundation and enterprise architecture
- Backend: FastAPI application scaffold with modular architecture
- Frontend: React 19 + Vite + TypeScript scaffold
- Docker: Development and production compose configurations
- Database: PostgreSQL + SQLAlchemy + Alembic setup
- Logging: Structured logging with rotation
- Error handling: Global exception handlers and custom exceptions
- Security: CORS, security headers, rate limiting architecture
- Documentation: Architecture, development guide, API standards
- CI/CD: GitHub Actions workflows
- Code quality: ESLint, Prettier, Black, Ruff, isort, MyPy, pre-commit
- VS Code workspace and Dev Container configuration
- Monitoring preparation: Prometheus, Grafana, Sentry stubs
